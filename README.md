## Routes
`./app/routes.js`

|Route|HTTP Verb|Description|
|-----|:-------:|----------|
|/api/posts|GET|Get all the posts.|
|/api/posts|POST|Create a post.|
|/api/posts/:id|GET|Get a single post.|
|/api/posts/:id|PUT|Update a post with new info.|
|/api/posts/:id|DELETE|Delete a post.|
